# Решение: дуальный код к коду Хэмминга -- симплексный код

## Постановка задачи

Проверить (программно), что код, дуальный коду Хэмминга, действительно является симплексным.

Симплексный код -- код, в котором все ненулевые кодовые слова имеют одинаковый вес, и расстояние между любыми двумя различными словами одинаково.

---

## 1. Теоретическое обоснование

### Дуальный код к коду Хэмминга

Код Хэмминга с параметром r имеет проверочную матрицу H размера r x (2^r - 1), столбцы которой -- все ненулевые двоичные векторы длины r.

Дуальный код: G_дуальный = H_Хэмминга. Параметры:

    n = 2^r - 1,  k = r

Ожидаемые свойства симплексного кода:

    Вес всех ненулевых слов = 2^(r-1)
    Расстояние между любыми двумя различными словами = 2^(r-1)
    Число ненулевых слов = 2^r - 1

### Почему это верно (теоретический аргумент)

Кодовое слово c = m * H, где m -- ненулевой вектор длины r. Элемент c_j = <m, v_j>, где v_j -- j-й столбец H.

Для ненулевого m функция f(v) = <m, v> на GF(2)^r имеет ядро размерности r-1, поэтому ровно 2^(r-1) ненулевых векторов дают f(v) = 1. Значит w(c) = 2^(r-1).

## 2. Программа (Python)

```python
import numpy as np
from itertools import product

def build_hamming_H(r):
    """Проверочная матрица кода Хэмминга: столбцы = числа 1..2^r-1 в двоичном виде."""
    n = 2**r - 1
    H = np.zeros((r, n), dtype=int)
    for col, i in enumerate(range(1, 2**r)):
        for row in range(r):
            H[r - 1 - row, col] = (i >> row) & 1
    return H

def check_simplex(r):
    """Проверка для заданного r."""
    H = build_hamming_H(r)
    G_dual = H  # порождающая матрица дуального кода
    n = 2**r - 1
    k = r
    expected_weight = 2**(r - 1)

    # Генерируем все кодовые слова
    weights = []
    for m in product([0, 1], repeat=k):
        c = np.array(m, dtype=int) @ G_dual % 2
        w = int(np.sum(c))
        weights.append(w)

    nonzero_weights = [w for w in weights if w > 0]
    is_simplex = all(w == expected_weight for w in nonzero_weights)
    return is_simplex, nonzero_weights, expected_weight

for r in [2, 3, 4, 5]:
    ok, weights, ew = check_simplex(r)
    n = 2**r - 1
    print(f"r={r}: ({n},{r})-код, ожидаемый вес={ew}, "
          f"фактические веса={sorted(set(weights))}, "
          f"симплексный: {'ДА' if ok else 'НЕТ'}")
```
