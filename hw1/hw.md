## ДЗ 2. Расстояние Хэмминга -- метрика

d(x, y) = число позиций, где x_i != y_i. Нужно показать 4 аксиомы:

1) d(x, y) >= 0 -- количество элементов множества неотрицательно.

2) d(x, y) = 0 <=> x = y -- нет различий <=> слова совпадают.

3) d(x, y) = d(y, x) -- "x_i != y_i" равносильно "y_i != x_i".

4) d(x, z) <= d(x, y) + d(y, z) -- неравенство треугольника:

   Пусть x_i != z_i. Тогда либо x_i != y_i, либо y_i != z_i
   (если x_i = y_i, то из x_i != z_i следует y_i != z_i).
   Значит множество различий (x, z) содержится в объединении
   различий (x, y) и (y, z), откуда d(x, z) <= d(x, y) + d(y, z). QED

## ДЗ 3. Код с d_min исправляет t <= floor((d_min - 1) / 2) ошибок

Доказательство (от противного).

Передано c1, произошло s <= t ошибок, принято y. Тогда d(y, c1) = s <= t.

Допустим, существует c2 != c1 такое что d(y, c2) <= t.
По неравенству треугольника:

    d_min <= d(c1, c2) <= d(c1, y) + d(y, c2) <= t + t = 2t

Но t <= floor((d_min - 1) / 2) означает 2t <= d_min - 1,
то есть d_min <= 2t <= d_min - 1 -- противоречие.

Значит c1 -- единственное ближайшее кодовое слово, декодирование верно.
