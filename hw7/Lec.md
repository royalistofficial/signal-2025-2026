# Конспект лекции 7: Решётки кодов, циклические коды и алгебраические основы

## 1. Методы построения решёток кодов
Существует три основных способа построения решёток для кодов:
* **По кодовым словам:** Решётка строится ярус за ярусом путем перечисления всех кодовых слов. Этот метод работает для любых кодов (не обязательно линейных), но имеет экспоненциальную сложность вычислений $O(2^k \cdot n)$.
* **По порождающей матрице $G$:** Применимо только для линейных кодов. Матрица приводится к минимальной спановой форме (МСФ), при которой все начала (begin) и концы (end) строк различны. Узлы на ярусах нумеруются значениями информационных символов активных строк. Сложность построения составляет $O(k \cdot n)$. 
* **По проверочной матрице $H$ (Синдромная решётка):** Узлы каждого яруса нумеруются накопленными (частичными) синдромами. Движение по решётке начинается с нулевого синдрома $000$ и обязательно заканчивается в узле $000$, поскольку кодовое слово, умноженное на матрицу $H$, всегда дает ноль.
* **Минимальность:** Решётки, построенные из МСФ или по проверочной матрице, являются минимальными. Это означает, что если кодовые слова имеют одинаковое «будущее», они обязательно проходят через один и тот же узел.

## 2. Циклические коды и полиномиальное представление
Циклические коды — это подкласс линейных блоковых кодов, в которых циклический сдвиг любого кодового слова также является кодовым словом.
* **Полиномиальное представление:** Кодовые слова представляются в виде полиномов (например, слово $10001$ соответствует $1 + x^4$). 
* **Циклический сдвиг:** Эквивалентен умножению полинома на $x$ по модулю двучлена $x^n - 1$.
* **Приведенный полином:** Полином, у которого коэффициент при старшей степени равен единице. При работе с двоичным полем $GF(2)$ все полиномы являются приведенными.
* **Порождающий полином $g(x)$:** Среди всех кодовых слов циклического кода существует единственный приведенный полином наименьшей степени. Любое кодовое слово делится на $g(x)$ без остатка.
* **Построение кодов:** Порождающий полином $g(x)$ всегда является делителем $x^n - 1$. Для построения циклического кода длины $n$ необходимо разложить $x^n - 1$ на множители над полем $GF(2)$ и выбрать их комбинацию в качестве $g(x)$.
* **Проверочный полином $h(x)$:** Получается при делении $x^n - 1$ на $g(x)$. Умножение любого кодового слова на $h(x)$ дает 0 по модулю $x^n - 1$.

## 3. Теоретико-алгебраические основы: Группы, Кольца, Поля
Для понимания устройства конечных полей необходимо рассмотреть базовые математические структуры:

### Группы
Множество с одной определенной операцией называется группой, если выполняются следующие аксиомы:
1.  **Замкнутость:** Результат операции над элементами множества принадлежит этому же множеству.
2.  **Ассоциативность:** Независимость от расстановки скобок при выполнении операций.
3.  **Единичный элемент:** Существует нейтральный элемент, применение операции с которым оставляет исходный элемент без изменений (например, пустая строка для конкатенации или 0 для сложения).
4.  **Обратный элемент:** Для каждого элемента существует обратный, дающий в результате операции единичный элемент.
* Если операция коммутативна (порядок операндов не влияет на результат), группа называется **абелевой (коммутативной)**.
* **Смежные классы и Фактор-группа:** Если подмножество $H$ само является группой, оно называется подгруппой. Прибавление элемента из основной группы к элементам подгруппы $H$ образует *смежный класс*. Смежные классы либо полностью совпадают, либо не пересекаются. Сами смежные классы образуют группу, называемую *фактор-группой*.

### Кольца
* Кольцо — это множество, на котором определены две операции (сложение и умножение).
* По сложению кольцо является абелевой группой.
* Относительно умножения выполняется замкнутость и ассоциативность. 
* Операции связаны дистрибутивностью.
* Главное отличие кольца: в нем может не быть обратных элементов по умножению, поэтому **деление в кольцах в общем случае невозможно**.

### Поля
* Поле — это кольцо, в котором **все ненулевые элементы образуют абелеву группу по умножению**. В поле возможно осуществлять деление (кроме деления на ноль).
* Множество чисел по модулю $q$ образует поле $GF(q)$ только в том случае, если $q$ — простое число (например, $GF(2)$ или $GF(5)$).

## 4. Кольцо многочленов и алгебраическое расширение полей
Чтобы получить конечное поле заданного размера (когда число элементов не является простым), применяется метод алгебраического расширения.
* **Кольцо многочленов:** Множество полиномов, коэффициенты которых берутся из поля $GF(q)$. Такое множество образует кольцо, где можно выполнять сложение, вычитание и умножение.
* **Алгебраическое расширение:** Выбирается **простой (неприводимый)** полином $p(x)$ степени $m$. Простой полином нельзя разложить на множители над заданным полем.
* Берется множество всех полиномов степени строго меньше $m$.
* Все операции над этим множеством производятся **по модулю неприводимого полинома $p(x)$** (берется остаток от деления).
* Поскольку $p(x)$ неразложим, каждый ненулевой многочлен имеет обратный элемент, и полученная структура автоматически становится полем.
* В результате формируется расширенное поле Галуа $GF(q^m)$, содержащее ровно $q^m$ элементов. (По этому же принципу поле действительных чисел расширяется до комплексных чисел через неприводимый многочлен $x^2 + 1$).
